= kifutter

みんなのつぶやきを集計して、Twitter上の募金総額を計算するよ

== 依存ライブラリ

* Sinatra
* Haml
* Sequel
* sqlite3-ruby
* twitterstream

== 起動方法

=== setup

  $ sudo gem install 依存ライブラリ

=== Create a db.

  $ sequel -m db/migration sqlite://db/kifutter.db

=== Start twitter watch.

  $ ruby watcher.rb

=== Start web app.

  $ ruby start.rb

http://localhost:4567/
