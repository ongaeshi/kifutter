= kifutter

みんなのつぶやきを集計して、Twitter上の募金総額を計算するよ

== 依存ライブラリ

* Sinatra
* Haml
* Sequel
* sqlite3-ruby

== 起動方法

=== setup

  $ sudo gem install sinatra haml sequel sqlite3-ruby

=== Create a db.

  $ sequel -m db/ sqlite://kiffutter.db

=== Start twitter watch.

  $ ruby watcher.db

=== Start web app.

  $ ruby start.rb

http://localhost:4567/
